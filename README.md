## Setup Instructions
Follow these steps to get the project up and running on your local machine.

Prerequisites
- Docker
- Docker Compose
- Node js

1. After cloning, do : 
```
npm install
```

2. Build and run the Docker containers:
```
docker-compose up --build
```

3. Run the app
```
node app.js
```

4. If success run the app, you can open `http://localhost:3000`
